# 🚦 Traffic Accident Analysis

## 📌 Project Overview
This project analyzes traffic accident data using Python to identify accident trends, high-risk locations, time-based patterns, and factors contributing to road accidents. The analysis helps understand accident behavior and supports road safety improvements.

## 🎯 Objectives
- Analyze accident trends over time
- Identify accident-prone locations
- Compare accidents by district and road type
- Analyze accidents by vehicle type
- Study accident severity and causes
- Generate business and safety insights using data

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- VS Code

## 📂 Dataset
- Tamil Nadu Traffic Accident Dataset (2021–2026)
- Contains accident details, location, vehicle type, severity, date, and road information.

## 📊 Analysis Performed
- Data Cleaning
- Missing Value Check
- Duplicate Value Check
- Descriptive Statistics
- Accident Distribution by District
- Monthly Accident Trend
- Vehicle-wise Accident Analysis
- Road Type Analysis
- Accident Severity Analysis
- Time-based Accident Analysis

## 📈 Key Insights
- Certain districts recorded significantly higher accident counts.
- Most accidents occurred during peak traffic hours.
- Two-wheelers were involved in a large percentage of accidents.
- National and State Highways experienced more severe accidents.
- Seasonal variations affected accident frequency.

## 🚀 Conclusion
This project demonstrates how Exploratory Data Analysis (EDA) can be used to identify accident patterns, understand risk factors, and support data-driven decisions for improving road safety and traffic management.
